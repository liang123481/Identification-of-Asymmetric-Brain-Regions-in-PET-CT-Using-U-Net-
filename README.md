# Identification-of-Asymmetric-Brain-Regions-in-PET-CT-Using-U-Net

本研究系湖南省大学生创新项目，已按照相应流程，在湖南省人民医院完成了 PET/CT
影像数据的收集与筛选，同时根据疾病类型（是否患有脑部疾病）及病情严重程度对影像
资料进行分类整理，最终纳入 52 例患者的 PET/CT 影像及对应临床病例信息（已隐去患
者个人隐私信息）。在此基础上，依据研究需求将数据集划分为训练集与测试集：分割经室
1 研究方法 3
间孔的横断面上下共三个断面的训练集包含 77 例影像, 主要用于 U-Net 模型对中间低摄
取度区域的分割训练, 分割断面中的颅骨部分训练集包含 66 例影像，主要用于 U-Net 模型
对颅骨的分割训练；测试集包含 11 例影像，主要来源于存在左右脑标准化摄取值（SUV）
差异的患者, 用于验证模型的有效性、适用性及最终结果的可视化呈现


本研究提出了一种结合 U-Net 融合策略与传统图像处理的 PET/CT 脑部不对称区域识
别方法。通过“原始图像 + 灰度图像预测交集”的融合分割框架，在训练集 77 例与测试集
25 例数据上实现了脑部非病灶区精准遮挡与四区域皮层划分，Dice 系数中位数达 0.859，同
时训练颅骨分割模型，对与后续分析无关的颅骨区域实现分割，Dice 系数中位数达 0.9806，
较传统方法提升明显；在此基础上，镜像翻转、重心粗对齐与 ECC 精细配准实现左右脑区
匹配，结合 SUV 差异热力图分析，剔除背景干扰并高亮代谢差异显著区域，最终生成的热
力图可直观、清晰地反映脑部示踪剂摄取存在显著差异的脑区。

U-net的训练同原U-net论文训练相同，我们添加一个代码供组预测，即批量预测。

512 原图转灰度批量.py
读取指定文件夹内的图片，先通过自定义 RGB 通道权重公式将彩色图转换为灰度图（针对性突出特定颜色的边界差异），再采用 CLAHE（对比度受限自适应直方图均衡化）算法增强灰度图的局部对比度，强化边界细节清晰度，最终将处理后的灰度图批量保存至指定输出文件夹。
交集扩展代码.py

批量处理两个预测掩码文件夹中的同名掩码文件：首先基于对应原图的 RGB 色彩相似度，对每个掩码进行局部区域扩展（将颜色相近的像素纳入掩码前景）；随后对扩展后的两个掩码执行像素级交集运算（仅保留两者共同的前景区域），最终生成并保存高精度的融合后二值掩码至指定文件夹。

前额四区域分割.py
读取原始图像及两个掩码文件，先对掩码执行相减运算（剔除重叠区域）；再自动检测大脑中轴线（基于对称度最优原则）及上部分割凹圆弧，将图像精准划分为左上、右上、左下、右下四个区域；最后分别保存各区域对应的掩码图像和原始图像，输出至指定路径。

原图和掩码拉伸 512x512.py
批量处理两个文件夹中同名的图片与掩码文件：按原图宽高比等比例缩放，通过填充黑色边框将尺寸统一至 512×512（全程无拉伸变形）；掩码复用对应原图的缩放、补边参数，确保与原图像素级对齐；最终将处理后的图片和掩码分别保存至指定输出路径。

原图对比度提升代码.py
批量遍历指定文件夹内的图片，先计算每张图片的平均亮度并动态适配 CLAHE 增强参数（低亮度图片增大增强幅度，高亮度图片降低增强幅度）；仅对 HSV 色彩空间的亮度（V）通道执行自适应直方图均衡化，在不改变色彩的前提下提升整体对比度，最终保存增强后的图片。

同部位翻转 SUV 差异热力图生成.py
批量匹配左右侧大脑图像：先将右侧图像水平翻转，与左侧图像执行 ECC（增强相关性系数）配准对齐；再计算配准后左右脑图像的灰度差异，生成差异热力图；随后过滤背景黑色区域优化热力图（仅保留有效脑区差异）；最终将左图、右图、处理前 / 后热力图按 1 行 4 列无间距紧凑排版，保存为可视化对比图，直观呈现左右脑差异。

灰度图对比度提升.py
批量处理指定文件夹内的图片：全程采用传统图像处理方法（无机器学习依赖），先通过自定义 RGB 权重转换为灰度图（突出特定颜色边界），再经 CLAHE 算法增强局部对比度；可选启用 Sobel 算子边缘检测模块，进一步强化边界细节；最终将处理后的灰度图批量保存至指定文件夹。

热力图生成.py
批量处理左右侧脑图像：先通过安全读取机制加载图片，将右侧图像水平翻转后，依次执行质心粗配准、ECC 算法精细配准（配准失败则降级为粗配准）；计算配准后左右脑的灰度差异并生成热力图，过滤黑背景区域优化差异展示；最终将左图、右图、处理前 / 后热力图按 1 行 4 列紧凑排版保存，清晰对比左右脑区域差异。
